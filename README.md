# Baobab Vision

Baobab Vision is a mobile application designed to enhance the eyewear shopping experience. Using augmented reality and facial shape classification, the app allows users to virtually try on eyeglasses and receive intelligent style recommendations tailored to their facial features.

This project leverages Flutter for the frontend and Node.js with MongoDB for the backend, providing a seamless and interactive user experience across Android and iOS platforms.

## 🌟 Key Features
- 📱 Augmented Reality (AR) virtual try-on
- 👓 Eyewear recommendations
- 📊 User face shape analytics
- 🔐 Secure authentication and user management
